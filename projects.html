About the Project

This character was created for the final project of the 3D Asset Pipeline for Games class. Unlike previous assignments, we were given free rein to make whatever we wanted to (as long as it fit into the fantasy theme).

My Contributions

This mushroom frog was a completely original design, beginning as a notebook sketch. I modeled the character in ZBrush and textured it in Substance Painter. The skeleton was provided for us, so the next step was painting the weights. A fully-animated template character controller was also provided to us in Unreal Engine 5, we just had to create an IK Rig and retarget the animations.

About the Project

This graphics engine was built over the course of a semester for the Game Graphics Programming class. We learned how to load 3D objects, apply texture and light to those objects using shaders, create cameras, render a sky, and implement PBR. All of this was done with the goal of simulating the techniques used in modern game engines.
